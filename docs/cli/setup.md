# `foxfang setup`

Initialize `~/.foxfang/foxfang.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- CLI onboarding: [Onboarding (CLI)](/start/wizard)

## Examples

```bash
foxfang setup
foxfang setup --workspace ~/.foxfang/workspace
```

To run onboarding via setup:

```bash
foxfang setup --wizard
```
