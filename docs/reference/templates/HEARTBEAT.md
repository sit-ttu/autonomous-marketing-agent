# HEARTBEAT.md

Keep this file empty or comments-only to skip proactive heartbeat work.

Example marketing checks to uncomment and adapt:

```markdown
# - Drafts or reports waiting for approval?

# - Content due in the next 24-48 hours?

# - Recurring report or metric review due?

# - Campaign metric anomaly in provided/read-only data?

# - Confirmed learning to add to MEMORY.md?

# - Open brief missing information from the operator?
```

If nothing needs attention, reply `HEARTBEAT_OK`.
